# 

過去に読んだ漫画やアニメを記録しておくプロダクトです。

## 扱っている技術

- React

フロントエンド:React18

- Laravel

バックエンド:Laravel9

- Vite

ビルドツール

- Inertia

ReactとLaravelをレンダリングされたページを動的に更新してくれるフレームワーク

-  TypeScript

型推論してくれる言語

- Recoil

Reactの状態管理ツール

- i18next

多言語化フレームワーク

- axios

非同期

- SQlite

リレーショナルデータベース管理システム

- Styled-Components

CSS in JSのライブラリ

- BootStrap

CSSフレームワーク

- Carbon

PHPの日付操作を簡単にするライブラリ
